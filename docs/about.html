About

What this is and how to customize it.

Dark Wiki is a minimal, static template intended for GitHub Pages. It avoids a build step, keeps markup simple, and uses accessible, high-contrast colors tuned for dark mode.

Customize

  • Edit assets/css/styles.css to tweak colors and layout.
  • Add pages under articles/; link them in the sidebar list.
  • Replace assets/favicon.svg with your own icon.
